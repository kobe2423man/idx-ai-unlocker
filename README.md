# 🚀 Project IDX AI Unlocker

专为 **Google Project IDX** 环境设计的一键脚本。  
只需一行命令，即可自动部署 Cloudflare Argo 隧道，完美解锁 **ChatGPT**、**Gemini**，并修复手机端 Shadowrocket 连接 AI 失败的问题。

---

## ✨ 核心功能

- 🌍 **免配置**：无需域名，自动集成 Cloudflare Tunnel  
- 🔓 **AI 全解锁**：完美解锁 ChatGPT、OpenAI、Gemini（解除地区限制）  
- 📱 **手机端修复**：服务端强制屏蔽 UDP，彻底解决 iOS / Shadowrocket 无法连接 Google 的问题  
- 🛡️ **防风控**：Google 流量全走 WARP，降低账号异常风险
